Introduction Last updated: 2022-10-01

pyQA is a framework built using selenium and pytest to simplify test automations. It has built in pre-written functions, which use selenium as its backbone, all the test cases run with a pre-customized pytest conftest file.

Motivation

Learning all new concepts and working of tools, their structure their syntax, it was time consuming, and so was manually testing products. We needed a something which was simple to use and with comparatively less code. Selenium is one of the best and a powerful option for web automation, and Pytest can be used to write various types of software tests, including unit tests, integration tests, end-to-end tests, and functional tests. Instead of diving deep in all the tools associated with automation testing, we wanted simple to use commands which are easy to understand even for a beginner programmer or a new person on a project.

Solution

pyQA was created to solve all the issues we stated above. It has pre written built-in functions, which use Selenium as backbone. To run all tests we use Pytest. Also as Playwright offered test with multiple browsers at once, pyQA also offers this feature. This highly customisable and scalable framework offers: simple code and file structure and endless customizability.
